# Araçlar · Tools

Manifesto'nun **kod, yazılım, web uygulamaları** ile geliştirilebileceği klasör. Her şey CC0 — kullan, çatalla, değiştir, dağıt.

## 🇹🇷 Türkçe

### Ne tür projeler eklenebilir?

- 📖 **Web okuyucu** — manifestoyu tarayıcıda güzel formatla göster
- 🔍 **Arama aracı** — tüm çeviriler içinde arama
- 📚 **E-book üretici** — EPUB, MOBI, PDF formatlarında otomatik export
- 🎙️ **Sesli kitap** — manifestoyu okuyup paylaşan ses dosyaları
- 🌐 **Çeviri hizalayıcı** — yan yana iki dili karşılaştırma aracı
- 🤖 **Bot** — Discord, Telegram, Twitter için günlük alıntı paylaşan bot
- 🧩 **Tarayıcı eklentisi** — okurken anında not alma, vurgulama
- 📱 **Mobil uygulama** — manifesto okuma uygulaması
- 📊 **Veri analizi** — kelime sıklığı, tema analizi, görselleştirme

### Format kuralları

- Her proje kendi alt klasöründe
- Klasör adı: `kebab-case` (örn: `web-reader`, `epub-builder`)
- İçinde **kendi README.md**'si olsun (nasıl kurulur, nasıl çalışır)
- Hangi dil/teknoloji: özgürce seç (Python, JavaScript, Rust, Go, vs.)
- Lisans: **CC0** (manifestoyla aynı — kamu malı)

## 🇬🇧 English

### What can be added?

- 📖 **Web reader** — display the manifesto beautifully in browser
- 🔍 **Search tool** — search across all translations
- 📚 **E-book generator** — auto-export to EPUB, MOBI, PDF
- 🎙️ **Audiobook** — voice recordings of the manifesto
- 🌐 **Translation aligner** — side-by-side comparison tool
- 🤖 **Bot** — Discord, Telegram, Twitter daily-quote bots
- 🧩 **Browser extension** — annotate while reading
- 📱 **Mobile app** — manifesto reader app
- 📊 **Data analysis** — word frequency, theme analysis, visualization

### Format rules

- Each project in its own subdirectory
- Folder name: `kebab-case`
- Include its own **README.md** (setup + usage)
- Any language/tech allowed (Python, JS, Rust, Go, etc.)
- License: **CC0**

### Suggested first project

**Static web reader** — a simple SSG that takes everything in `manifesto/translations/` and produces a beautiful unified website. Jekyll, Hugo, Astro, MkDocs — your choice.
