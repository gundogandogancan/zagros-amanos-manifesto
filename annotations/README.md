# Açıklamalar · Annotations

Manifesto üzerine kişisel, akademik, eleştirel yorumlar.

## Format

- Dosya adı: `yazarAdı-konu.md` (örn: `mehmet-tarihselBaglam.md`)
- Markdown formatında
- Manifesto'dan alıntı yaparken bölüm referansı ver
- Lisans: CC0 (katkın da kamuya armağan olur)

## Türler

- 📚 **Akademik** — kaynak referanslı analiz
- ✍️ **Eleştirel** — karşı görüş, sorgulama
- 💭 **Kişisel** — deneyim, anı, yorum
- 🔍 **Karşılaştırmalı** — başka eserlerle ilişkilendirme
- 🌍 **Tarihsel** — manifesto'nun referans verdiği döneme dair derinleştirme
