## Başlamak için / To start

1. Issue aç (Translation template)
2. Bu klasörde `[dil-kodu]/` klasörü oluştur (örn: `en/`, `ku/`, `ar/`)
3. `00-preface.md` ile başla
4. CONTRIBUTING.md'ye bak
