# Sanat & Medya · Art & Media

Manifesto'ya görsel, sesli, video veya müzikal katkılar.

## Alt klasörler / Subdirectories

```
assets/
├── illustrations/   ← Çizim, resim, görsel sanat
├── audio/           ← Sesli kitap, müzik, ortam sesleri
├── video/           ← Kısa filmler, animasyon, dokümanter
├── photography/     ← Zagros, Amanos, Mezopotamya fotoğrafları
└── typography/      ← Font, kaligrafi, yazı tasarımları
```

## Format kuralları

- Dosya adı: `yazarAdı-konu-yyyymmdd.ext` (örn: `aysel-zagrosGunDogusu-20260605.jpg`)
- Yüksek çözünürlük tercih edilir
- Telifsiz olmalı (CC0 ile uyumlu) veya **kendi eserin** olmalı
- Her klasörde bir `CREDITS.md` dosyası — kim ne yapmış listesi
